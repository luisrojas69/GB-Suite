# GB-Suite
GB.Suite es un conjuto de herramientas desarrolladas a medida para GB

Modulos:

Pozos.
	- Listado de Pozos (Exportable a excel)
	- CRUD Completo.
	- Dashboard
	- Aforos por pozo 
	- CRUD de Aforos
	- Mantenimientos Correctivos
	- CRUD Completo de mattos


Areas.
	- CRUD Sectores
	- CRUD Lotes
	- CRUD tablones (codigo autogenerado)
